How to fix it


Make sure the eslint and node commands are on the PATH.
  1. Go to TextMate > Preferences… > Variables
  2. Ensure the PATH is enabled there and that it includes the location of your eslint and node commands.

How to disable validation


If you mistakenly installed this validation tool and want to disable it, you can do so in TextMate:

  1. On the TextMate menu, choose Bundles > Edit Bundles…
  2. Locate “JavaScript ESLint”
  3. Uncheck “Enable this item”
  4. Close the Bundle Editor and choose “Save”
